## Agent skills

### Issue tracker

Issues 以 Markdown 文件形式存放在本仓库的 `.scratch/` 下（每功能一个目录）。See `docs/agents/issue-tracker.md`.

### Triage labels

使用五个默认 triage 标签：`needs-triage` / `needs-info` / `ready-for-agent` / `ready-for-human` / `wontfix`。See `docs/agents/triage-labels.md`.

### Domain docs

Single-context 布局：仓库根目录的 `CONTEXT.md` + `docs/adr/`。See `docs/agents/domain.md`.
